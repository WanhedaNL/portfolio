# Cesim Portfolio

Persoonlijke portfolio website van Cesim Bayram.

## Live

[cesimbayram.github.io/portfolio](https://cesimbayram.github.io/portfolio)

## Technologieën

- HTML
- CSS
- JavaScript (Vanilla)

## Functies

- Navigatie met smooth scroll
- Scroll animaties (fade-in)
- Projecten overzicht
- Contactformulier (Formspree)
- Responsive design (mobiel vriendelijk)
- Back to top knop
